# Project Setup
1. Change `python_version` to `{ your_python_version }` (like `3.7`) @ `./Pipfile`
2. Setup environment
    ```
    rm Pipfile.lock
    pip install pipenv
    pipenv shell
    pipenv install
    ```
3. Open file: `./src/genetic_algorithm.ipynb` or `./src/particle_swarm_optimization.ipynb`
4. Run all cell @ `opened file`
